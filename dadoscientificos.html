Dados cientificos do big data

1. Volume de Dados Gerados Diariamente: Estima-se que mais de 2,5 quintilhões de bytes de dados sejam gerados a cada dia no mundo. Esse volume inclui dados de dispositivos, redes sociais, sensores, transações financeiras, e-mails e muito mais.

2. Variedade de Dados: Os dados são altamente diversos em termos de formato e tipo. Isso inclui dados estruturados (como bancos de dados tradicionais), dados semiestruturados (como documentos XML) e dados não estruturados (como imagens, áudio e vídeo).

3. Velocidade de Geração de Dados: Em algumas aplicações, os dados são gerados em tempo real, exigindo análises e tomada de decisões em tempo quase real. Isso é especialmente importante em campos como finanças, monitoramento de saúde e IoT (Internet das Coisas).

4. A quantidade total de dados deve chegar a 79 ZB: A infraestrutura digital emergente somada aos hábitos de utilização de soluções computacionais estão contribuindo fortemente para o volume cada vez maior de dados. Estima-se que até o final de 2021 a quantidade total de dados no mundo chegue a 79 zetabytes. Um zetabyte é aproximadamente igual a um bilhão de terabytes.

5. A infraestrutura de streaming e análise de dados vai aumentar cinco vezes até 2024: Isso decorre do fato de que 75% das organizações passarão da fase piloto para a operacionalização ou implantação de inteligência artificial até 2024. As soluções analíticas avançadas fornecem insights e soluções vitais para as organizações, e a utilização só aumentará nos próximos anos.

6. Até 2022, 35% das grandes organizações participarão de mercados formais de dados online: O crescimento de volumes de dados é uma prova de como eles são valiosos para as organizações de todos os setores. Não é surpresa, então, que um mercado formal de dados online deva surgir nos próximos anos, com uma fração das grandes organizações atuando neste mercado.

7. Entre 80% e 90% dos dados no universo digital não são estruturados: A maioria dos dados não é estruturada, estando disponibilizada na forma de áudio, vídeo, imagens e texto. Ferramentas sofisticadas de análise de dados, como análise de texto e análise de sentimentos, reconhecimento de padrões e conversão de voz em texto, são necessárias para extrair insights nestes tipos de dados. É importante ainda destacar que isso é uma realidade e necessidade no contexto da pesquisa em mídias digitais.

8. Cerca de 60% a 70% dos dados corporativos não são usados: A maioria dos dados disponíveis nas organizações permanece subutilizada, dado que o processo de conversão de dados brutos em insights úteis e valiosos é de grande complexidade. As organizações não precisam apenas investir em tecnologias analíticas, mas também em uma cultura baseada em dados, que esteja presente em todos os departamentos, assim como no desenvolvimento de produtos data-driven.

9. Cerca de 20% das organizações em todos os setores têm implantado big data: Apesar do crescente volume de dados e das inúmeras oportunidades apresentadas por esse crescimento, apenas algumas organizações estão operando big data em vários casos funcionais. Entretanto, o big data está se tornando um tema importante na agenda dos executivos. Organizações que ainda não adotaram a tecnologia estão expressando interesse em usos futuros. Um relatório de 2019 mostra que a análise de big data ocupa a 26ª posição em uma lista de 37 tecnologias vitais para a inteligência de negócios.
